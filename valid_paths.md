## Valid Paths ✅

- Forward slashes (recommended):
```bash
CSV_PATH="C:/Users/user/Documents/data.csv"
```

- Double backslashes (recommended):
```bash
CSV_PATH="C:\\Users\\user\\Documents\\data.csv"
```

- Without quotes:
```bash
CSV_PATH=C:\Users\user\Documents\data.csv
```
---
---
## Inavlid Path ❌

- Single backslashes with quotes:
```bash
CSV_PATH = "C:\Users\user\Documents\data.csv"
```
